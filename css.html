Introdução ao CSS

O CSS (Cascading Style Sheets) é uma linguagem de estilo usada para controlar a apresentação e a aparência de elementos HTML em uma página da web. Ele permite que você defina cores, fontes, espaçamento, layout e muito mais para tornar suas páginas web visualmente atraentes e consistentes.

Principais Conceitos do CSS

Aqui estão alguns conceitos fundamentais do CSS que você deve conhecer:

  • Seletor: Um seletor é usado para identificar o elemento HTML ao qual você deseja aplicar estilos. Pode ser o nome de uma tag HTML, uma classe ou um ID.
  • Propriedade: Uma propriedade é um atributo que você deseja definir para um elemento, como cor de fundo, tamanho da fonte ou margem.
  • Valor: O valor é o que você atribui a uma propriedade. Por exemplo, você pode definir a cor de fundo como "verde" ou o tamanho da fonte como "16px".
  • Declaração CSS: Uma declaração CSS consiste em um seletor, uma propriedade e um valor, separados por dois-pontos e terminados por ponto-e-vírgula.
  • Folha de Estilo Externa: É possível criar uma folha de estilo CSS separada em um arquivo externo e vinculá-la ao seu documento HTML usando a tag <link>.

O CSS permite que você selecione elementos HTML e aplique estilos a eles de forma eficiente, o que torna suas páginas web mais flexíveis e fáceis de manter.

Exemplo de Estilização CSS

Aqui está um exemplo simples de como você pode usar CSS para estilizar um elemento HTML:

        
            <style>
                h1 {
                    color: blue;
                    font-size: 24px;
                }
            </style>

Neste exemplo, estamos definindo que todos os elementos `<h1>` na página terão uma cor azul e um tamanho de fonte de 24 pixels.

Agora que você tem uma breve introdução ao CSS, você está pronto para começar a estilizar suas páginas web e torná-las visualmente atraentes!
